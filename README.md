# The Drop

A monorepo containing the frontend and backend for The Drop — a news app.

## Structure

```
the-drop/
├── frontend/   # React/Vite frontend (the-drop-news)
├── backend/    # Python backend API (thedrop-app)
└── README.md
```

## Frontend

Located in `/frontend`. A React + Vite application with Tailwind CSS.

```bash
cd frontend
npm install
npm run dev
```

## Backend

Located in `/backend`. A Python backend service.

```bash
cd backend
# See backend/README.md for setup instructions
```
